# Горячие клавиши
#### Выделить всё:
Ctrl+A
#### Копиравать:
Ctrl+C
#### Вставить:
Ctrl+V
#### Сохранить:
Ctrl+S
#### Вырезать:
Ctrl+X
#### Отменить:
ctrl+Z

# работа с fit
### скачать проект:
```bash
git clone <URL>
```
### учесть все файлы:
```bash
git add .
```
### Сохранить коммит :
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
### Принять изменения из облака:
```bash
git pull
```
